# Feature Additions Design — 2026-04-05

## 목표

저장된 스냅샷 데이터를 더 풍부하게 분석/시각화하고,
종목 클릭 시 Alpha Vantage API를 통해 현재 시세와 차트를 앱 내에서 확인할 수 있도록 한다.

---

## 기능 1: 기간별 수익률 요약 카드

### 위치
히스토리 섹션(자산 변화 라인 차트 아래) 하단에 새 카드 추가.

### 동작
- 주간 / 월간 탭 전환 UI
- localStorage에 저장된 스냅샷 데이터를 기반으로 계산 (외부 API 없음)
- **주간**: ISO 캘린더 기준 월~일 주간 내 첫 스냅샷 → 마지막 스냅샷 기준으로 수익률 변화, 평가금액 변화 표시
- **월간**: 각 월의 첫 스냅샷 → 마지막 스냅샷 기준 동일 계산
- 해당 기간 스냅샷이 1개뿐이면 변화량은 "—"로 표시
- 스냅샷이 2개 미만인 경우 카드 자체를 숨김

### 표시 항목 (기간별 행)
- 기간 레이블 (예: "2026년 3월", "3월 4주차")
- 수익률 변화 (+X.Xp%)
- 평가금액 변화 (+XXX만원)
- 시작 수익률 → 끝 수익률

## 기능 2: 종목 상세 모달 (Alpha Vantage 연동)

### 트리거
종목 리스트의 각 종목 행 클릭 시 모달 팝업.

### 모달 구성
1. **헤더**: 종목명, 현재가, 당일 등락률
2. **차트**: 최근 30일 종가 라인 차트 (Chart.js)
3. **티커 입력**: 티커 코드 미등록 종목은 입력 필드 표시 후 저장

### 티커 코드 관리
- 종목명 → 티커 코드 매핑을 localStorage(`tickerMap`)에 저장
- 처음 클릭 시: "이 종목의 티커 코드를 입력해주세요 (예: 005930.KS)" 안내 후 입력
- 저장 후 자동으로 API 호출하여 차트/시세 표시
- 등록된 종목은 이후 바로 모달 열림

### API 호출
- 엔드포인트: Alpha Vantage `TIME_SERIES_DAILY` (최근 30일)
- 현재가: `GLOBAL_QUOTE` 엔드포인트
- 호출 실패 또는 API 키 미설정 시 에러 메시지 표시 ("현재 시세를 불러올 수 없어요")
- 무료 티어 제한(25회/일) 초과 시 동일 메시지 처리

## 기능 3: API 키 설정

### 위치
앱 헤더 우측에 설정 아이콘(⚙) 추가.

### 동작
- 클릭 시 드롭다운 또는 인라인 패널 표시
- Alpha Vantage API 키 입력 필드 + 저장 버튼
- 저장 시 localStorage(`avApiKey`)에 보관
- "무료 키 발급 → alphavantage.co" 링크 표시
- 키가 없는 상태에서 종목 모달 열면 "API 키를 먼저 설정해주세요" 안내

---

## 데이터 저장 구조

| 키 | 내용 |
|---|---|
| `portfolioSnapshots` | 기존 스냅샷 배열 (변경 없음) |
| `tickerMap` | `{ "삼성전자": "005930.KS", ... }` |
| `avApiKey` | Alpha Vantage API 키 문자열 |
